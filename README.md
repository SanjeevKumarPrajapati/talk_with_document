**Talk_With_Document** is an **AI-powered document chatbot** that allows users to **ask questions and interact with any uploaded document** using natural language.  
The system extracts text from documents, builds a **semantic index**, and provides **intelligent, context-aware answers** strictly grounded in the document content.

It supports PDFs, Word files, and text documents, making it ideal for **learning, research, and quick information retrieval**.
